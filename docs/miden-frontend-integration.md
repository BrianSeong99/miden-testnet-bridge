# Miden Frontend Integration

The canonical Miden wallet and Next.js UI guidance now lives in
[`frontend/docs/miden-frontend-integration.md`](../frontend/docs/miden-frontend-integration.md).

That document is the source of truth before changing:

- MidenFi wallet adapter behavior,
- Miden SDK or WASM loading behavior,
- wallet-launched receive/send flows,
- Miden note claim or consume UX,
- AggLayer, NEAR Intents mock, or Epoch route copy.

This root-level file is kept as a stable pointer for backend docs, wallet-chat
docs, and older links. Do not duplicate implementation guidance here; update the
frontend doc instead.
