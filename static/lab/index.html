Outbound: Miden to Anvil

sequenceDiagram
    participant UserClient
    participant BridgeAPI as Bridge API
    participant Miden
    participant Anvil
    UserClient->>BridgeAPI: POST /v0/quote miden-testnet:eth -> eth-anvil:eth
    BridgeAPI-->>UserClient: Stable bridge account + BridgeOutV1 memo
    UserClient->>Miden: Create public BridgeOutV1 note
    BridgeAPI->>Miden: Sync and validate public note
    BridgeAPI->>Miden: Consume note with bridge account
    BridgeAPI->>Anvil: Release ETH to recipient
    BridgeAPI-->>UserClient: GET /v0/status = SUCCESS

Anvil to Miden

Inbound P2ID mint

Requests a 1Click-shaped quote, sends native ETH on Anvil, waits for the bridge to mint a solver-signed public P2ID note on Miden, then lets the recipient claim it.

Miden to Anvil

Public BridgeOutV1 note

Funds a Miden wallet through the inbound route, then submits a public programmable bridge note. The bridge consumes the note and releases native ETH on Anvil.
